Utility methods:

Although using these methods is not necessary nor required, you can use them in all challenges, through the U object.


- clamp (x, min, max)
Returns x clamped between min and max. clamp(-1, 0, 1)=0, clamp(2, 0, 1)=1.

- lerp1d (from, to, x)
Returns the linear interpolation between from and to with factor x. If from=0 and to=2 and x=0.5, it returns 1.

- lerp2d (from, to, x)
The same as lerp1d but from and to are 2-dimensional arrays. with from=[0,0] and to=[2,0] and x=0.5, it returns [1,0]

- vectorSum (a,b)
Element-by-element sum of arrays of vectors length. vectorSum([1,1],[2,0])=[3,1].

- vectorSobtract (a,b)
Element-by-element difference of vectors of same length.

- vectorMultiply (a,b)
Element-by-element multiplcation of vectors of same length.

- distance (a,b)
Euclidean distance between two vectors of size 2.

- manhattanDistance (a,b)
Manhattan distance between two vectors of size 2.

- transpose (v)
Transposes a vector, returning its inverse. transpose([1,2,3]) = [3,2,1].
